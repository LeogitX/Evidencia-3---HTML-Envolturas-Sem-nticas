# Ejercicio 14 - Parte 3: Página de Empresa (empresa.html)

### Instrucciones para visualizar:
1.  Asegúrate de tener la extensión Live Server instalada en VS Code.
2.  Haz clic derecho en el archivo `src/index.html` en el explorador de archivos.
3.  Selecciona "Open with Live Server".
4.  Navega a la página de la empresa usando el enlace correspondiente.

## Objetivo
Crear una página corporativa simple utilizando envolturas semánticas de HTML5.

## 🎯 Conceptos a evaluar:
- Uso de `<header>`, `<nav>`, `<main>`, `<section>`, `<aside>`, `<footer>`.
- Estructura semántica para una página de empresa.

### Documentación
- [HTML Semántico](https://developer.mozilla.org/es/docs/Glossary/Semantics#semntica_en_html)

## Instrucciones
1.  Dentro de la carpeta `src/ejercicio-14`, crea un archivo llamado `empresa.html`.
2.  **Estructura semántica requerida**:
    -   `<header>`: Logo o nombre de la empresa y navegación.
    -   `<nav>`: Menú de navegación consistente.
    -   `<main>`: Contenido principal.
    -   `<section>`: Al menos 3 secciones (ej. Servicios, Acerca de, Contacto).
    -   `<aside>`: Información adicional o testimonios.
    -   `<footer>`: Pie de página consistente con información de contacto.

### Estructura de carpetas esperada:
```
src/ejercicio-14/
├── index.html
├── blog.html
└── empresa.html
```
